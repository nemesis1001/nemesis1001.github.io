### [리눅스(Linux)] root권한, vi command not found 해결

> 버전정보  
> OS: ubuntu 16.04 LTS

- root 패스워드 변경하기
	- $ sudo passwd root
- root으로 접속하기
	- $ su
- root에서 logout 방법
	- $ exit
- PATH설정을 하거나 vim 설치
	- $ sudo apt-get update
	- $ sudo apt-get install vim
- vim으로 vi 대체해서 사용하기
